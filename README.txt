Description

  Epic Chat is a light-weight, free and open source web based chat
  application with multi-tabbed chatting, realtime audio and video
  upload, transliteration and predictive language processing.


Features

  - Multi-tabbed chatting 
  - Realtime audio and video upload
  - Transliteration with Google Transliterate
  - Smileys
